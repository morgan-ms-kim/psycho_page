# test1: 오락실 캐릭터 유형 테스트

## 개요
- 레트로 오락실 감성의 캐릭터 유형 심리테스트
- 10~20대가 좋아할만한 키치&큐트 스타일

## 예시 문항
1. 오락실에 가면 가장 먼저 하는 일은?
   - a) 게임기 구경하기
   - b) 동전 바꾸기
   - c) 인기 게임 찾기
   - d) 친구와 사진 찍기
2. 가장 좋아하는 게임 장르는?
   - a) 리듬게임
   - b) 격투게임
   - c) 퍼즐게임
   - d) 스포츠게임

## 결과 예시
- A타입: 열정적인 챌린저
- B타입: 전략가
- C타입: 감성러
- D타입: 사교왕
